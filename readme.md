# CodeTest - ReactNative

## File

|--- rnApps (react native application)

## Development

1. install dependence

```bash
cd rnApps
yarn install
```

2. start ios

```bash
cd ios
pod install
cd ..
npx react-native run-ios
```

3. start android

```bash
npx react-native run-android
```
